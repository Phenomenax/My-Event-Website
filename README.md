# World of Flavors - International Food Festival

A responsive event website for an international food festival, built as part of the DECO7140 course.

## Overview

A simple website promoting the "World of Flavors" event.

## Features

- **Home** – Event introduction with discount code generator
- **Gallery** – Photo showcase of cuisines
- **About** – Live weather widget, newsletter subscription form

## Tech Stack

- HTML5, CSS3, JavaScript
- Weather API integration
- Google Fonts (Poppins)

## Getting Started

Open `src/index.html` in a browser to view the site.

## Project Structure

```
src/
├── index.html          # Home page
├── gallery.html        # Image gallery
├── about.html          # About & subscribe
├── css/                # Stylesheets
├── js/                 # Scripts & API calls
└── images/             # Site assets
```
